**스마트포인터(Smart Pointer)**

**개요**
-스마트포인터는 C++에서 사용할 수 있는 특수한 포인터이다.
-스마트포인터는 객체의 메모리 할당 및 해제를 자동으로 처리하기 위해 사용된다.
-스마트포인터는 포인터와 동일한 기능을 제공하면서, 객체 생성 및 파괴에 대한 책임을 가지고 있으며, 메모리 해제를 자동으로 처리하기 위해 만들어진 포인터이다.

**장점**
- 자동 메모리 해제: 메모리에 할당된 객체의 생명주기가 종료되면 자동으로 해제함에 의해 메모리 해제를 수행하므로, 개발자가 직접 메모리 해제를 처리할 필요가 없다.
- 메모리 누수 방지: 객체의 생명주기가 종료되면 자동으로 메모리 해제를 수행하므로, 메모리 누수를 방지할 수 있다.
- 객체의 생명주기 추적: 객체의 생명주기(생성, 사용, 파괴)를 추적하여, 어떤 객체가 언제 생성되었고 언제 파괴되었는지 확인할 수 있다.
- 코드 가독성 향상: 자동 메모리 해제를 수행하므로, 코드가 가독성이 향상된다.

**단점**
- 성능 저하: 자동으로 메모리를 할당 및 해제 해주기 위해 기존의 메모리 할당 및 해제보다 시간이 더 걸릴 수 있다.
- 복잡성: 포인터를 추적하고 메모리를 해제하기 위해 코드가 복잡해질 수 있다.
- 호환성 문제: C++98/03 에서는 기본적으로 제공되지 않는다. C++11에서 지원되기 시작한다.

**스마트포인터 종류**
- unique_ptr: 각각의 객체에 대해 하나의 포인터만 접근할 수 있다.
- shared_ptr: 여러 포인터가 하나의 객체를 가리킬 수 있다.
- weak_ptr: 객체를 참조할 수 있지만, 객체를 수정할 수 없다.

**활용**
- 객체의 생성 및 파괴를 자동으로 처리하기 위해 사용할 수 있다.
- 메모리 누수를 방지하기 위해 사용할 수 있다.
- 객체의 생명주기 추적하기 위해 사용할 수 있다.
- 코드 가독성 향상을 위해 사용할 수 있다.
