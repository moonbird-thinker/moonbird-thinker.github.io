**콕스 상품(Cox Model)**

콕스 상품은 루돌프 콕스(Rudolph Cox)가 1972년에 제안한 생존 분석 모델입니다. 콕스 상품은 시간 의존적인 위험 함수를 가정하지 않으며, 간단한 형태와 명확한 해석 가능성으로 인해 생존 분석에서 가장 널리 사용되는 모델 중 하나입니다.

**콕스 상품의 상세 설명**

콕스 상품은 다음과 같이 표현됩니다.

```
h(t) = h_0(t) * exp(β_1 * x_1 + β_2 * x_2 + ... + β_p * x_p)
```

여기서,

* h(t)는 시간 t에서의 위험 함수입니다.
* h_0(t)는 기본 위험 함수입니다.
* β_1, β_2, ..., β_p는 독립 변수 x_1, x_2, ..., x_p의 회귀 계수입니다.

기본 위험 함수 h_0(t)는 일반적으로 미정함수로 취급되며, 추정할 때는 특정 함수 형태를 가정합니다. 가장 일반적으로 사용되는 기본 위험 함수 형태는 지수 함수와 와이블 함수입니다.

**장점:**

* 비례 가정을 만족할 필요가 없음

**단점:**

* 개별적인 생존 함수를 추정할 수 없음
* 독립 변수의 효과가 시간에 따라 변하는 경우에는 적합하지 않음
